# Bitcoin Self-Custody Lab
비트코인 노드 운영, 홈 마이닝, UTXO 관리 및 자체 보관(Self-custody) 설정에 대한 실험과 기록을 정리하는 저장소입니다.

## 다루는 주제

- 비트코인 노드 설정
- 홈 마이닝
- UTXO 관리
- 자체 보관 워크플로
- 지갑 복구 연습
- 프라이버시 및 보안 관련 기록

## 목적

이 저장소는 비트코인 자율 운용 과정에서 실제로 경험한 설정, 시행착오, 운영 원칙을 기록하기 위한 공간입니다.

## 현재 기록 예정 항목

- Umbrel / Bitcoin Core 운영 기록
- Sparrow Wallet 연결 및 사용 기록
- 홈 마이닝 로그
- UTXO 분리 및 관리 원칙
- 복구 훈련 및 보안 점검 메모

## 주의

이 저장소의 내용은 개인 실험 및 학습 기록이며  
투자 조언이나 완전한 보안 지침을 제공하는 목적이 아닙니다.
