# Bitcoin-self-custody-Lab
비트코인 노드 운영, 홈 마이닝, UTXO 관리 및 자체 보관 설정에 대한 실험과 기록을 정리하는 저장소입니다

## 다루는 주제

- Bitcoin node setup
- Home mining
- UTXO management
- Self-custody workflows
- Wallet recovery practice
- Privacy and security notes

## 목적

이 저장소는 비트코인 자율 운용 과정에서 실제로 경험한 설정, 시행착오, 운영 원칙을 기록하기 위한 공간입니다

## 현재 기록 예정 항목

- Umbrel / Bitcoin Core 운영 기록
- Sparrow Wallet 연결 및 사용 기록
- 홈 마이닝 세팅 로그
- UTXO 분리 및 관리 원칙
- 복구 훈련 및 보안 점검 메모

## 주의

이 저장소의 내용은 개인 실험 및 학습 기록이며 투자 조언이나 보안 완전성을 보장하지 않습니다
